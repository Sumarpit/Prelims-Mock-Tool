# UPSC Prelims Mock Tool

A clean, web-based mock test tool designed for UPSC CSE Prelims.

## Features
- **UI:** Clean, distraction-free interface (Segoe UI).
- **Automation:** Upload a PDF to the `/uploads` folder, and GitHub Actions automatically converts it to a JSON test file.
- **PWA:** Can be installed as an app.

## How to Add a Test
1. Go to the `uploads/` folder in this repository.
2. Upload your **Forum IAS SFG PDF**.
3. Wait 30 seconds for the GitHub Action to run.
